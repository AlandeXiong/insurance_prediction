IBM Watson Marketing: Auto Insurance Renewal/Response Prediction

This is a classic, feature-rich dataset for insurance renewal/repurchase response prediction. It contains policy details, claim history, and customer demographic attributes.

Dataset highlights:

- Customer profile: gender, education, marital status, income, employment status.
- Policy details: effective date, coverage tier (Basic/Extended/Premium), premium, policy type.
- Claims history: months since last claim, total claim amount.
- Customer behavior: complaints, renewal offer response, vehicle class/size.

Source: “Customer Value Data - Marketing Analysis” (IBM sample dataset used widely in analytics demos).
