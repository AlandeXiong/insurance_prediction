IBM Watson 营销：汽车保险续保预测
这是目前最经典、维度最全的保险续保数据集。它包含了完整的保单细节、理赔历史以及客户的人口统计学特征。

数据集特点：

客户信息： 年龄、性别、教育程度、婚姻状况、收入、就业状态。

保单信息： 保单生效日期、覆盖范围（Basic/Extended/Premium）、保费、保单类型。

理赔信息： 上次理赔后的月数、理赔总额（Total Claim Amount）。

用户行为： 投诉次数、续保提议（Renewal Offer）响应情况、车辆类型及行驶里程。
